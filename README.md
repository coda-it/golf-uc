# golf-uc
This repository is designated for developing a handheld device based on NodeMCU along with OLED screen.

### Hardware:
* NodeMCU on ESP-12E
* OLED blue 1,3" 128x64px I2C v2 SH1106

### Dependencies:
* https://github.com/olikraus/u8g2/
